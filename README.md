# QA & UX 로그

서비스 QA 진행 중 발견한 이슈와 UX 개선 방향을 기록한 문서입니다.  
본 문서는 실제 서비스 QA 경험을 기반으로 하되, 특정 회사·서비스·화면·데이터가 식별되지 않도록 내용을 일반화하고 재구성했습니다.

## 목적
- QA 경험 및 사고 과정 기록
- 사용자 흐름, 상태 관리, 데이터 정합성, validation 이슈 분석
- 기획자 관점 문제 정의

## 케이스 작성 형식
| 항목 | 내용 |
|------|------|
| 문제 | 어떤 문제가 있었는가 |
| 기대 동작 | 사용자가 기대하는 흐름 |
| 실제 동작 | 실제로 발생한 상황 |
| 개선 방향 | 기획/UX/개발 관점 제안 |

## 대상 서비스 유형
스포츠 대회 참가, 라운딩 기록, 실시간 스코어 관리부터 심판/운영자 기능을 포함한 
스포츠 경기 운영 올인원 플랫폼 (모바일 앱 + 고객용 관리자 + 운영 관리자 웹)

- 대회 관리 (신청/참가/이력)
- 라운딩 관리
- 실시간 스코어 입력 및 순위
- 심판 기능
- 관리자 운영 기능
- 결제 및 포인트 시스템
